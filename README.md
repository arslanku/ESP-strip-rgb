## ✨ Возможности

- **Поддержка нескольких платформ**: ESP32 и ESP8266 (с настройкой через макросы)
- **MQTT управление**: Полный контроль через топики (вкл/выкл, яркость, режимы)
- **7+ световых эффектов**: Радуга, костёр, дискотека, одноцветные режимы и многое другое
- **OTA обновления**: Беспроводная прошивка без USB-кабеля
- **Плавное включение/выключение**: Эффект затухания и разгорания
- **Автоматическое переподключение**: При потере WiFi или MQTT соединения
- **Конфигурация под разные ленты**: Количество светодиодов, пин подключения, тип ленты

## 🛠 Аппаратное обеспечение

### Поддерживаемые платы
- ESP32 (любая плата с GPIO5)
- ESP8266 (NodeMCU, Wemos D1 Mini и аналоги)
- Легко расширяется под другие платы через макросы

### Подключение ленты
| Плата | DATA PIN |
|-------|----------|
| ESP32 | GPIO5 |
| ESP8266 | D1 (GPIO5) |

### Необходимые компоненты
- Светодиодная лента на базе WS2812/WS2812B (или аналогичная)
- Блок питания 5V (мощность зависит от длины ленты)
- При необходимости — уровень-конвертер (если лента 5V, а логика 3.3V)

## 🚀 Начало работы

### 1. Клонирование репозитория
```bash
git clone https://github.com/arslanku/esp32-table-rgb.git
cd esp32-table-rgb
```

### 2. Настройка PlatformIO
Откройте проект в [PlatformIO](https://platformio.org) (VS Code + расширение PlatformIO).

### 3. Конфигурация
Создайте файл `include/auth_data.h` (на основе `auth_data_example.h`):
```cpp
// WiFi настройки
#define WIFI_SSID "your_wifi_name"
#define WIFI_PASSWORD "your_wifi_password"

// MQTT настройки
#define MQTT_SERVER "mqtt.your-server.com"
#define MQTT_PORT 1883
#define MQTT_USER "your_mqtt_user"
#define MQTT_PASSWORD "your_mqtt_password"
```

### 4. Выбор платы
В `main.cpp` выберите нужный контроллер:
```cpp
#define CONTROLLER_TYPE_ESP32_TABLE 1     // Для ESP32
// #define CONTROLLER_TYPE_ESP8266_TABLE 1 // Для ESP8266 (стол)
// #define CONTROLLER_TYPE_ESP8266_WINDOW 1 // Для ESP8266 (окно)
```

### 5. Сборка и загрузка
- **Через USB**: Выберите среду `esp32dev` или `esp8266dev` и нажмите Upload
- **Через OTA**: После первой загрузки используйте среду с суффиксом `_ota`

## 📡 MQTT топики

### Подписка (управление)
| Топик | Значения | Описание |
|-------|----------|----------|
| `STRIP-table-power` | `0` / `1` | Включение/выключение ленты |
| `STRIP-table-brightness` | `0`-`100` | Яркость в процентах |
| `STRIP-table-mode` | `RAINBOW`, `CAMPFIRE`, `RED`, `GREEN`, `BLUE`, `DISCO` | Выбор режима |
| `STRIP-table-speed` | `low` / `medium` / `high` | Скорость смены эффектов в дискотеке |
| `STRIP-table-1-OTA` | `1` | Перезагрузка контроллера |

> **Примечание**: Названия топиков меняются в зависимости от выбранного контроллера (table-1, table-2, window).

### Публикация (обратная связь)
- `STRIP-table-1-received` — подтверждение получения команд

## 🎨 Световые эффекты

| Режим | Описание |
|-------|----------|
| **RAINBOW** | Классическая бегущая радуга по всей ленте |
| **CAMPFIRE** | Имитация костра с мерцающими тёплыми оттенками |
| **RED** / **GREEN** / **BLUE** | Одноцветная заливка (статика) |
| **DISCO** | Дискотека с 6 автоматически сменяемыми подэффектами: |
| | 0. Быстрые радужные вспышки |
| | 1. Быстрая радуга (вперёд) |
| | 2. Бегущие огни |
| | 3. Быстрая радуга (назад) |
| | 4. Случайные цветные сектора |
| | 5. Пульсирующая волна цвета |

## 📦 OTA обновления

Проект поддерживает беспроводные обновления прошивки.

### Первая загрузка (через USB)
```bash
pio run -e esp32dev -t upload   # для ESP32
# или
pio run -e esp8266dev -t upload # для ESP8266
```

### Последующие загрузки (через WiFi)
```bash
pio run -e esp32dev_ota -t upload --upload-port <IP_адрес>
```

IP-адрес устройства отображается в Serial Monitor после подключения к WiFi.

## ⚙️ Конфигурация

### Основные параметры (в `settings.h`)
```cpp
#define NUM_LEDS 120              // Количество светодиодов
#define DATA_PIN 5                // Пин подключения
#define LED_TYPE WS2812            // Тип ленты
#define COLOR_ORDER GRB            // Порядок цветов
#define LOGGING 1                  // Включить логирование в Serial
```

### Настройка эффектов (в `strip_control.h`)
- `discoTimerMs` — скорость смены эффектов в дискотеке
- Параметры пульсации в функции `pulsatingBrightness()`

## 🔧 Устранение неполадок

| Проблема | Решение |
|----------|---------|
| Не подключается к WiFi | Проверьте SSID и пароль в `auth_data.h` |
| Не подключается к MQTT | Проверьте адрес сервера, порт, логин/пароль |
| Лента не светится | Проверьте пин DATA_PIN и питание ленты |
| Неправильные цвета | Измените `COLOR_ORDER` (GRB/RGB) |
| Мигает странно | Проверьте блок питания (возможно, не хватает тока) |
| Не работает OTA | Убедитесь, что компьютер и ESP в одной сети |
